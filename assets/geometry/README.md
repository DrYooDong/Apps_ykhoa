# CliniPortal Geometry & Shapes Library

Bộ sưu tập các hình khối hình học, đường phân tách section (dividers) và các mẫu trang trí SVG/CSS phục vụ layout y khoa chuyên nghiệp.

## 📁 Cấu trúc thư mục

```
geometry/
├── geometry-shapes.css   # Thư viện hình dạng CSS & Dividers
├── demo-geometry.html    # Trang xem trước các hình khối & Dividers
└── README.md             # Tài liệu này
```

## 🔷 Thành phần chính

1. **Section Dividers (Đường phân tách)**:
   - Các đường lượn sóng (`wave-divider`), xiên chéo (`slant-divider`), răng cưa (`zigzag-divider`) làm mềm các điểm nối tiếp.

2. **Medical Geometric Patterns (Nền hình học)**:
   - Hoa văn lưới điểm (`dot-grid`), họa tiết xoắn kép DNA, lưới tế bào hỗ trợ tạo cảm giác hiện đại, khoa học.

3. **Isometric Equipments (Hình chiếu trục đo 3D)**:
   - Các hình hộp, bục trưng bày 3D cho Infographics.
