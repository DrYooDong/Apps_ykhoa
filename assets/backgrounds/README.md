# CliniPortal Background Effects Library

Thư viện hiệu ứng hình nền bao gồm Gradients y khoa, chuyển động nền mượt và hiệu ứng Glassmorphism hỗ trợ tối đa cho cả chế độ Sáng (Light) và Tối (Dark).

## 📁 Cấu trúc thư mục

```
backgrounds/
├── background-effects.css   # Thư viện màu nền & hiệu ứng chuyển động
├── demo-backgrounds.html    # Bản xem thử hiệu ứng nền tương tác
└── README.md                # Tài liệu này
```

## 🌊 Các dạng hình nền chính

1. **Medical Gradients (Chuyển màu y khoa)**:
   - Các dải màu được tối ưu hóa như: `bg-cardiovascular` (đỏ tim mạch), `bg-nephrology` (xanh da trời nhạt), `bg-pharmacology` (tím nhạt).

2. **Animated Backgrounds (Nền động)**:
   - Các hiệu ứng di chuyển nhẹ nhàng của bong bóng khí hoặc dải sáng chéo tạo chiều sâu mà không gây phân tâm.

3. **Glassmorphism Blur Overlays**:
   - Các lớp phủ làm mờ nội dung phía sau nhằm tăng mức độ tập trung cho các Modals và Popups.
