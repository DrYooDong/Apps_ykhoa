Interactive Forest Plot Builder

Công cụ trực quan hóa kết quả tổng quan hệ thống (Meta-analysis) và phân tích nhóm (Subgroup Analysis). Tự động vẽ biểu đồ Forest Plot chuẩn bằng đồ họa SVG sắc nét, tính toán chỉ số dị biệt I² và hình thoi gộp (Pooled Estimate Diamond).

Biểu Đồ Forest Plot SVG Tương Tác:

Biểu đồ cập nhật thời gian thực theo bảng dữ liệu phía dưới. Kích thước hình vuông thể hiện Trọng số (Weight %), khoảng gạch ngang thể hiện 95% CI và hình thoi biểu thị Hiệu quả gộp.
